ALTErNATE ACryLIC SySTEMS

Odorless Acrylic System

• Same application as Advanced Retention Acrylic System (See Page 7). • Use only soft color acrylic powders, clear and natural.

• Use drier mixing ratio: remove all liquid from one side of brush and half from the other • Product sets slower and yields softer enhancements. • File off tacky layer before fnishing.

Odor-Free UV Cured Acrylic System

• Same application as Odorless Acrylic System. (See above). • Use only soft color acrylic powders, clear and natural.

• Use drier mixing ratio: remove all liquid from one side of brush and half from the other • Product sets under a Nouveau Nail Mini Tunnel UV Lamp in 1-3 minutes. • File off tacky layer before fnishing.

Quick Dip™ Acrylic System

• 10-Step Quick Prep — Apply nail tips only. (See Page 5).

• Brush a coat of Brush-On Wrap Gel over entire nail, avoiding cuticle area.

• Dip entire nail straight into Quick Dip™ Acrylic Powder . Tap off any excess powder. • Repeat the last two steps. Add a third and fnal coat of Brush-On Wrap Gel.

• Mist nails with Super Fast Set Activator Spray ™. Nails should be dry within seconds.

• Use a Nouveau Nail Washable 100/180 File to shape the free edge and straighten the side walls. • Follow steps 10-13 for French Acrylic Application (See Page 8).

• Finish with Nouveau Spa Lacquer ™, Glass Seal ™ or Super Seal ™ UV Gel Sealer .
